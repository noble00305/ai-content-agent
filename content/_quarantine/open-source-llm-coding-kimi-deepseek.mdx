## Ollama로 로컬에서 실행하기

### 사전 준비

| 요구사항 | 최소 | 권장 |
|----------|------|------|
| RAM | 16GB | 32GB+ |
| VRAM | 8GB | 24GB+ (RTX 4090 등) |
| 디스크 | 20GB 여유 | 50GB+ |
| OS | Windows/Mac/Linux | - |

### 설치 및 실행

```bash
# 1. Ollama 설치 (https://ollama.com)
# Windows: 설치 파일 다운로드
# Mac: brew install ollama
# Linux: curl -fsSL https://ollama.com/install.sh | sh

# 2. 모델 다운로드 및 실행
ollama pull kimi-k2.6        # Kimi K2.6
ollama pull deepseek-v4       # DeepSeek V4
ollama pull glm-5.1           # GLM-5.1

# 3. 실행
ollama run kimi-k2.6

# 4. API 서버로 사용 (포트 11434)
# 자동으로 REST API 제공
curl http://localhost:11434/api/generate -d '{
  "model": "kimi-k2.6",
  "prompt": "Python으로 이진 탐색 구현해줘"
}'
```

### 양자화 모델 선택 가이드

| VRAM | 권장 양자화 | 성능 손실 | 속도 |
|------|-----------|----------|------|
| 8GB | Q4_K_M | 중간 | 보통 |
| 16GB | Q6_K | 적음 | 빠름 |
| 24GB+ | Q8_0 또는 FP16 | 거의 없음 | 빠름 |

VRAM이 부족하면 양자화 모델을 쓰되, 코딩 작업에서는 **Q6_K 이상**을 권장한다. Q4 이하에서는 복잡한 로직 생성 시 정확도가 눈에 띄게 떨어진다.

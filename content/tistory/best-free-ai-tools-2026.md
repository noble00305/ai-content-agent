## 핵심 요약

- 글쓰기·이미지·코딩·영상·생산성 5개 용도로 나눠 무료 AI 도구 15개를 비교표로 정리했다 (2026년 7월 기준).
- 도구 하나를 고르는 것보다 **여러 도구의 조합**이 결과물을 좌우한다 — 용도별 추천 조합 포함.
- "무료"에는 대가가 있다: 입력 데이터의 학습 활용 가능성, 예고 없는 플랜 변경, 상업 이용 제한.

무료 플랜의 한도와 조건은 자주 바뀐다. 이 글의 수치는 모두 2026년 7월 기준이며, 각 항목에 출처를 달아 두었으니 실제 가입 전에 공식 가격 페이지를 한 번 더 확인하는 것이 안전하다.

## 1. 글쓰기 / 텍스트

| 도구 | 무료 범위 (2026년 7월 기준) | 강점 | 약점 |
|------|------|------|------|
| ChatGPT | GPT-5.x Instant 기본, 5시간당 약 10메시지 (초과 시 mini 계열 모델로 전환) | 범용성, 생태계 | 무료 한도 내 상위 모델 사용량이 적음 |
| Claude | 5시간 단위 사용량 제한 (대화 복잡도에 따라 대략 15~40메시지) | 긴 문서 분석, 자연스러운 문장 | 한도 도달이 빠른 편 |
| Gemini | 무료 (사용량 제한 있음) | 구글 검색·워크스페이스 연동 | 한국어 미묘한 뉘앙스는 아쉬움 |

ChatGPT 무료 플랜은 과거 "GPT-4o mini 무제한" 구조였지만, 지금은 최신 GPT-5.x Instant 계열을 기본으로 쓰다가 5시간 한도를 넘기면 경량(mini) 모델로 자동 전환되는 방식이다. 정확한 메시지 수는 공개 고정값이 아니라 상황에 따라 달라진다 ([OpenAI 무료 티어 FAQ](https://help.openai.com/en/articles/9275245-chatgpt-free-tier-faq)). Claude 무료 플랜도 "하루 몇 회"가 아니라 **첫 메시지 시점부터 5시간 창** 단위로 토큰 기반 한도가 적용된다 ([Claude 도움말](https://support.claude.com/en/articles/11647753-how-do-usage-and-length-limits-work)).

## 2. 이미지 생성

| 도구 | 무료 범위 (2026년 7월 기준) | 강점 | 약점 |
|------|------|------|------|
| Microsoft Copilot 이미지 생성 (구 Bing Image Creator) | 무료 (부스트 소진 시 느려짐) | DALL-E 계열 기반, 진입 장벽 없음 | 생성 속도 |
| Leonardo AI | 일 150토큰 | 스타일 프리셋 다양 | 토큰이 금방 소진됨 |
| Ideogram | 주 단위 무료 크레딧 (주 10 슬로우 크레딧 보고, 변동 이력 있음) | 이미지 속 텍스트 렌더링 | 포토리얼은 상대적으로 약함 |

Bing Image Creator는 Microsoft Copilot 브랜드로 통합됐다. Ideogram 무료 한도는 공식 문서상 명시가 유동적이고, 과거 "일 단위"에서 "주 단위"로 축소된 이력이 있어 확정 수치로 믿지 않는 게 좋다 ([Ideogram 플랜 문서](https://docs.ideogram.ai/plans-and-pricing/available-plans)). 무료 생성물이 커뮤니티에 공개된다는 점도 업무용이라면 체크할 부분.

## 3. 코딩

| 도구 | 무료 범위 (2026년 7월 기준) | 강점 | 약점 |
|------|------|------|------|
| GitHub Copilot | 월 2,000회 코드 완성 + 월 50회 채팅 | IDE 통합 | 무료 범위가 좁음 |
| Cursor | 무료 티어 (제한적 사용량) | AI 네이티브 에디터 | 무료 사용량이 적음 |
| Claude Code | 유료 구독 필요 (무료 아님) | 프로젝트 전체 맥락 이해 | 구독 전제 |

Copilot 무료 티어 한도는 [GitHub 공식 안내](https://github.com/features/copilot) 기준. Claude Code는 엄밀히 무료 도구가 아니지만, 코딩 카테고리 비교를 위해 포함했다 — "무료로 시작"은 Copilot이나 Cursor 쪽이다.

## 4. 영상 / 오디오

| 도구 | 무료 범위 (2026년 7월 기준) | 강점 | 약점 |
|------|------|------|------|
| CapCut | 기본 편집·일반 내보내기 무료, 워터마크 없음 | 자동 자막, 템플릿 | Pro 템플릿·일부 AI 기능 사용 시에만 워터마크 |
| ElevenLabs | 월 10,000크레딧 (고품질 TTS 약 10분 분량) | TTS 품질 | 무료는 비상업 용도 한정 |
| Suno | 일 단위 무료 크레딧 (약 10곡) | AI 작곡 | 무료 생성물 상업 사용 불가 |

CapCut에 대한 흔한 오해가 "무료면 워터마크가 박힌다"인데, 실제로는 **일반 편집 후 내보내기에는 무료 플랜에도 워터마크가 없다**. 워터마크가 붙는 건 Pro 표시가 있는 템플릿·효과·AI 기능을 무료로 썼을 때다 ([CapCut 공식 도움말](https://www.capcut.com/help/video-without-watermark)).

### ElevenLabs — 무료로 충분한가, 유료로 갈 시점인가

무료 플랜은 월 10,000크레딧이 매월 리셋되고 음성 클로닝 슬롯도 일부 제공되지만, **비상업 용도 한정**이다 ([ElevenLabs 가격 페이지](https://elevenlabs.io/pricing)). 판단 기준은 단순하다.

- **무료 유지**: 개인 학습, 내부 테스트, 월 10분 이내 나레이션이면 충분하다.
- **유료 전환 고려**: (1) 유튜브 등 수익화 콘텐츠에 음성을 쓰거나, (2) 월 나레이션이 10분을 넘거나, (3) 클로닝 음성을 상업 프로젝트에 써야 할 때. 이 세 가지 중 하나라도 해당하면 무료 플랜 약관 위반이거나 한도 초과이므로 Starter 이상이 필요하다.

## 5. 생산성

| 도구 | 무료 범위 (2026년 7월 기준) | 강점 | 약점 |
|------|------|------|------|
| Notion AI | 제한적 (AI 기능은 사실상 유료) | 문서+AI 통합 | 무료로는 맛보기 수준 |
| Perplexity | 무료 (Pro 검색은 일 단위 소량 제한) | 출처 달린 AI 검색 | 고급 모델은 Pro 필요 |
| Gamma | 가입 시 1회성 400크레딧 (AI 생성 약 8~10회, 매월 리셋 아님) | 프레젠테이션 자동 생성 | 크레딧 소진 후 AI 기능 사용 불가 |

### Gamma — 400크레딧의 정확한 의미

Gamma 무료 크레딧은 흔히 "10회 무료"로 알려져 있지만 정확히는 **가입 시 1회 지급되는 400크레딧**이고, 매월 충전되지 않는다. AI 생성 1건에 약 40크레딧이 들어 대략 8~10개 문서를 만들면 끝난다 ([Gamma 크레딧 안내](https://help.gamma.app/en/articles/7834324-how-do-credits-work-in-gamma), [가격 페이지](https://gamma.app/pricing)). 유료 전환 판단 기준:

- **무료 유지**: 발표 자료를 어쩌다 한 번 만들거나, 초안만 AI로 뽑고 나머지는 수동 편집하는 경우. 추천인 제도로 크레딧을 일부 보충할 수도 있다.
- **유료 전환 고려**: (1) 매달 정기적으로 AI 생성이 필요하거나, (2) 워터마크 제거·내보내기 형식이 업무상 필요하거나, (3) 팀 단위로 쓰는 경우. 월 수 회 이상 꾸준히 쓴다면 크레딧 재구매보다 구독이 단순하다.

## 용도별 선택 가이드

| 상황 | 1순위 | 이유 |
|------|------|------|
| 긴 보고서·문서 요약 | Claude | 긴 컨텍스트 처리 |
| 검색 기반 리서치 | Perplexity | 출처 표기 |
| 이미지에 텍스트 넣기 (배너·썸네일) | Ideogram | 텍스트 렌더링 |
| 비용 0원으로 이미지 최다 생성 | Copilot 이미지 생성 | 사실상 무제한 (느림 감수) |
| 영상 편집 입문 | CapCut | 일반 내보내기 워터마크 없음 |
| 한국어 나레이션 | ElevenLabs | 다국어 TTS 품질 |
| 발표 자료 초안 | Gamma | 문서→슬라이드 자동화 |

## 추천 조합

**블로그 글 작성:**
Perplexity(리서치) → Claude(초안) → ChatGPT(제목·메타 최적화) → Ideogram(썸네일)

**유튜브 쇼츠:**
ChatGPT(스크립트) → ElevenLabs(나레이션) → CapCut(편집+자막)

**코딩:**
Copilot(일상 자동완성) → ChatGPT 또는 Claude(디버깅·설계 논의), 프로젝트 단위 작업이 많다면 Claude Code 구독 검토

## 주의할 점

1. **무료 = 데이터 학습 대상일 수 있음** — 도구별 설정에서 학습 제외(opt-out) 여부를 확인하고, 민감 정보는 넣지 말 것.
2. **무료 플랜은 예고 없이 바뀐다** — Ideogram이 일 단위에서 주 단위로 축소된 것처럼, 핵심 워크플로우를 무료 플랜 하나에 의존하지 말 것. 이 글의 한도 수치도 시간이 지나면 달라질 수 있다.
3. **상업 이용 조건 확인** — ElevenLabs·Suno 등은 무료 생성물의 상업 사용이 제한된다. 수익화 콘텐츠라면 약관부터 볼 것.
4. **AI 결과물은 검증** — 팩트 기반 콘텐츠는 반드시 원출처를 확인할 것.

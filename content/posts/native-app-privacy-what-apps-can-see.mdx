## "허용"이 실제로 주는 권한 범위

| 권한 | 사용자가 생각하는 것 | 앱이 실제로 할 수 있는 것 |
|------|:---:|:---:|
| 카메라 | 사진 찍을 때만 | 앱 사용 중 언제든 촬영 가능 |
| 위치 | 지도 볼 때만 | 백그라운드에서 계속 추적 가능 |
| 연락처 | 친구 찾기용 | 전체 연락처 복사 가능 |
| 사진 | 올릴 사진 선택 | 전체 사진첩 접근 가능 |
| 마이크 | 통화할 때만 | 앱 사용 중 녹음 가능 |
| 건강 데이터 | 운동 기록 | 심박수, 수면, 생리주기까지 |

## AI 앱이 더 위험한 이유

1. **학습한다** — 일반 앱은 저장하고 표시. AI 앱은 패턴을 추출한다.
2. **클라우드로 전송** — 대부분 서버에서 처리. 데이터가 어디에 얼마나 보관되는지 불투명.
3. **제3자 공유** — "서비스 개선을 위해 제3자와 공유" 문구 있으면 광고사/데이터 브로커에게 넘어갈 수 있음.

## 자기 보호 체크리스트

**iOS:**
- 설정 → 개인정보 보호 → 각 항목에서 접근 앱 확인
- 위치: "항상 허용" → "사용 중에만"으로 변경
- 사진: "전체 사진" → "선택한 사진"으로 변경
- 앱 추적 투명성에서 추적 요청 차단

**Android:**
- 설정 → 보안 및 개인정보 → 권한 관리자
- 안 쓰는 앱 권한 해제
- 백그라운드 위치 접근 앱 확인

**AI 앱 사용시:**
- 어떤 데이터가 서버로 전송되는지 확인
- 개인정보처리방침에서 "데이터 보관 기간" 확인
- 가능하면 온디바이스 AI 앱 선택 (데이터가 폰을 안 떠남)
